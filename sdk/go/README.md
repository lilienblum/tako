# tako.sh Go SDK

Go SDK for [Tako](https://tako.sh) — deploy Go apps with zero configuration.

## Install

```bash
go get tako.sh
```

## Usage

### net/http

```go
mux := http.NewServeMux()
mux.HandleFunc("/", handler)
tako.ListenAndServe(mux)
```

### Gin

```go
r := gin.Default()
r.GET("/", handler)
tako.ListenAndServe(r) // *gin.Engine implements http.Handler
```

### Echo

```go
e := echo.New()
e.GET("/", handler)
tako.ListenAndServe(e) // *echo.Echo implements http.Handler
```

### Chi

```go
r := chi.NewRouter()
r.Get("/", handler)
tako.ListenAndServe(r) // chi.Mux implements http.Handler
```

### Fiber (fasthttp)

Fiber uses fasthttp instead of net/http, so use `Listener()` directly:

```go
app := fiber.New()
app.Get("/", handler)

ln, err := tako.Listener()
if err != nil {
    log.Fatal(err)
}
app.Listener(ln)
```

## Secrets

Secrets are accessed via a generated `Secrets` struct — no string keys, full autocompletion:

```go
db := Secrets.DatabaseUrl()
key := Secrets.ApiKey()
```

Generate (or update) the typed secrets file:

```bash
tako typegen
```

This reads secret names from `.tako/secrets.json` and generates `tako_secrets.go` with a typed `Secrets` struct. Run `tako typegen` after adding or removing secrets.

Generated code looks like this:

```go
// Code generated by tako typegen. DO NOT EDIT.
package main

import "tako.sh"

var Secrets = struct {
	ApiKey      func() string
	DatabaseUrl func() string
}{
	ApiKey:      func() string { return tako.GetSecret("API_KEY") },
	DatabaseUrl: func() string { return tako.GetSecret("DATABASE_URL") },
}
```

## API

### `tako.ListenAndServe(handler http.Handler) error`

Wraps the handler with Tako protocol and starts serving. Works with any `http.Handler` — Gin, Echo, Chi, gorilla/mux, net/http.

### `tako.Listener() (net.Listener, error)`

Returns a `net.Listener` for the Tako environment. Use this for frameworks that manage their own server (Fiber, custom http.Server).

### `tako typegen` (CLI command)

Generates `tako_secrets.go` with a typed `Secrets` struct. Run standalone — no dev server needed.

## How it works

`ListenAndServe` = `Listener()` + protocol wrapping + `http.Serve()`:

- **Production** (`--socket` flag): Unix socket for tako-server
- **Development** (no flags): TCP on `PORT` env var or `:3000`
- Intercepts `Host: tako` requests for health checks and secrets
- Forwards everything else to your handler

## Configuration

```toml
name = "my-app"
runtime = "go"
main = "app"

[build]
run = "CGO_ENABLED=0 go build -o app ."

[envs.production]
route = "my-app.example.com"
```

Deploy automatically cross-compiles for the target server (`GOOS=linux`, `GOARCH` from server config).
